Proyecto Java CRUD

Administrador de pedidos

- Para realizar pedidos necesita ingresar usuario y contraseña

- Puede registrarse y acceder o utilizar una de las cuentas ya creadas en la base de datos

- Una vez ingresó usuario y contraseña puede crear, buscar, modificar o eliminar pedidos

- Si no inició sesión o cerró sesión a través de otra pestaña no podrá realizar ningún pedido y se le enviará a la home page
